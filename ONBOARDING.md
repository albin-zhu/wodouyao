### Skills to Know About
- `/wodouyao` — Collaborate with peer terminals inside a Wodouyao canvas. Use when running inside a Wodouyao terminal (`WODOUYAO_ID` is set) to discover peers, spawn terminals, send keystrokes, or read peer output.
- `/skill-creator` — Scaffold new skills that extend Claude's capabilities with specialized knowledge or workflows. The team uses this to package reusable automations.
- `/skills` — Browse and manage the skills available in your environment.
- `/plugin` and `/reload-plugins` — Install and refresh Claude Code plugins when iterating on local skill or plugin development.
- `/resume` — Resume a previous Claude Code session to continue work with context intact.
